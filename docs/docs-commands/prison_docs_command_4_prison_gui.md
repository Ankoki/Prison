## Description:

This command will open the `/prison gui` where an admin can manage a lot of things about prison on a GUI.

## Permission:

- `prison.gui`
- `prison.admin`

## SubCommands:

- `none`

## How to use the command

Just execute the command himself `/prison gui`

### Command Format

`/prison gui`

**END of the command INFO**
